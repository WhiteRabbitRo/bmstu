1. на графике провести линию
Iк = 0 => Uкэ = Eк
Uк = 0 => Iк = Eк / Rк
2. взять середину, обозначить как А - рабочая точка, в проекциях на ось Uкэ найти 
Uкэ, в проекции на ось Iк найти Iк.
3. начертить изогнутую линию и найти по пропорции Iб0.
4. на маленький график перевести Iб0 на Iб, спроецировать на не нулевую ветку, 
спроецировать на Uбэ, получаем Uбэ0.
5. R1 = (Eк - Uбэ0) / Iб0.
6. R2 = (0.3 % 0.5) * R1 (любое число от 0.3 до 0.5).
7. h11э: на маленьком графике выберем две точки на Uбэ (+- 0.1В), прочертим 
проекции на Iбэ.
h11э =  прирощение (Uбэ) / прирощение (Iбэ).
8. h12э: на маленьком графике: приращение Uбэ = расстояние между проекцией А на 
Uбэ и проекцией пересечения проекии А на Iб и Uкэ=0.
приращение Uкэ = (Uкэ= %d) - (Uкэ = 0), %d - значение второй кривой на маленьком 
графике.
9. h21э: на большом графике: изменение Iк: провести линию Uк = Uке 
(через рабочую точку), через очки пересечений с соседними с нарисованной раннее 
изогнутой линией провести проекции на Iк, расстояние между ними - искомое 
изменение Iк.
изменение Iб = разница в подписях соседних с нарисованной ранее изогнутой линии 
изогнутых линий. 
h21э = изменение Ik / изменение Iб.
10. h22э: на большом графике, задаем случайную точку на Uк от 0 до Uкэ. Нахожим
разницу проекций случайной точки через нарисованную кривую по U и I. 
h22э = прирощение по Iк /  прирощение по Uк.
11. Rб = (R1*R2)/(R1+R2).
12. Rвх = (Rб*h11э) / (Rб+h11б). Определим входное сопротивление усилительного каскада, RВХ
13. Rвых = Rк / (1+h22э*Rк). Определим выходное сопротивление усилительного каскада.
14. Ku = h21э*Rk/(h11э*(1+h22э*Rr)). Определим коэффициент усиления по напряжению.
15. Ki = h21э*Rвых/(Rвых + Rk). Определим коэффициент усиления по току.
16. Kp = Ku*Ki. Определим коэффициент усиления по мощности.
17. Uвых = Ku*Uвх. Определим величину выходного напряжения усилительного каскада.
18. Rэ ~ (0.1 % 0.2)Eк/Iэ0  Iэ0 ~ Iк.
19. Сэ = 10 / (2*3,14*1000*Rэ). 
20. Eк = R1*Iб.
21. R1 = h21*Rk.
22. R2 = R1 / 4.
23. Rф = Ek - Uп / Ik.
24. Cф = 10 / 2*3,14*1000*Rф.
