```
gcd(a,b){
    if(b==0) return a;
return gcd(b,a%b);
}
```
```
fun gcd(a,b){
    while(b!=0)
    {
        int c;
        c = a;
        a = b;
        b = c % b;
    }
    return a;
}
```
